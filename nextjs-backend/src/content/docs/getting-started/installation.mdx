## Prerequisites

Before you begin, ensure you have the following installed:
- **Node.js 18+**
- **Python 3.12+**
- **pnpm** (Package Manager)
- **uv** (Python Package Manager)

## Step-by-Step Setup

<Steps>
  <Step>
    ### Clone the Repository
    ```bash
    git clone https://github.com/CubeStar1/ai-keyboard.git
    cd ai-keyboard
    ```
  </Step>
  <Step>
    ### Install Dependencies
    Install dependencies for all three layers:
    ```bash
    # Frontend (Electron)
    cd frontend && pnpm install
    
    # Next.js Backend
    cd ../nextjs-backend && pnpm install
    
    # Memory Backend (Python)
    cd ../backend && uv sync
    ```
  </Step>
  <Step>
    ### Configure Environment Variables
    Copy the example environment files and fill in your API keys (OpenAI, Supabase, Neo4j, etc.).
    ```bash
    cp frontend/env.example frontend/.env.local
    cp nextjs-backend/env.example nextjs-backend/.env
    cp backend/env.example backend/.env
    ```
  </Step>
  <Step>
    ### Database Setup
    - **Supabase**: Run the schema from `frontend/src/lib/supabase/migrations/schema.sql` in your Supabase SQL editor.
    - **Neo4j**: Set up a free AuraDB instance and save the credentials.
  </Step>
</Steps>

## Running the App

To run the full suite, you need to start four processes:

1. **Memory Backend**: `cd backend && uv run main.py`
2. **Next.js API**: `cd nextjs-backend && npm run dev`
3. **Windows MCP**: `cd frontend && npm run windows-mcp`
4. **Electron App**: `cd frontend && npm run dev`

The app will then be available in your system tray. Try pressing `Ctrl+\` to open the Action Menu.
