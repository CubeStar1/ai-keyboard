## What is Tabby?

Tabby is a **system-wide AI assistant** that lives at the point of input. Instead of switching between apps for AI help, it provides seamless integration through global shortcuts, ghost text suggestions, and desktop automation.

<Callout type="info">Tabby runs locally as a desktop application with Electron, connecting to powerful backend services for memory and AI processing.</Callout>

## Quick Start

import { Steps, Step } from 'fumadocs-ui/components/steps'

<Steps>
  <Step>
    ### Install Prerequisites
    Ensure you have Node.js, Python 3.12+, and pnpm installed on your system.
  </Step>
  <Step>
    ### Clone & Setup
    Follow the [Installation Guide](/docs/getting-started/installation) to clone the repo and set up environment variables.
  </Step>
  <Step>
    ### Launch Backend Services
    Start the Next.js API, Memory Backend, and Windows MCP server.
  </Step>
  <Step>
    ### Start the Desktop App
    Run the Electron application and look for the Tabby icon in your system tray.
  </Step>
</Steps>
