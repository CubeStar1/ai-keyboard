The Action Menu is a versatile tool for quick text transformations. It works anywhere you can select text—be it an IDE, a web browser, or an email client.

## Triggering the Menu

Press **Ctrl + \\** to open the Action Menu. If you have text selected, it will be automatically pulled into the context.

![Action Menu](/landing/tabby-actions.png)

## Included Actions

Tabby comes with a set of built-in actions to speed up your workflow:

- **Fix Grammar**: Instantly correct spelling and grammatical errors.
- **Shorten/Expand**: Adjust the length of your selected text.
- **Tone Adjustment**: Quickly change the tone to Professional, Casual, or Friendly.
- **Email Writer**: Draft emails based on basic bullet points.

## Quick Chat Mode

In the Action Menu, you can press **Tab** to switch to a quick chat mode. This allows you to give custom instructions like "Rewrite this in the style of Steve Jobs" or "Extract all dates from this text."

## Output Modes

You can choose how the AI's response is delivered:

1. **Paste (Default)**: The result is copied to your clipboard and pasted immediately.
2. **Typewriter**: The AI types out the result character-by-character. This mode is useful for avoiding automated text detection or for a more natural look.
